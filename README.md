# AXON

> **Offline AI Assistant powered by On-Device AI**

### *Your laptop remembers everything, so you don't have to.*

AXON is an AI-powered desktop assistant that runs **completely offline**. It enables users to chat with AI, search documents by their contents instead of filenames, perform intelligent research across local files, and interact with documents using natural language—all without sending any data to the cloud.

Whether you're a student, researcher, or professional, AXON helps you find information faster while keeping your data completely private.

---

## ✨ Features

- 🤖 Offline AI Chat powered by local language models
- 🔍 Content-Based File Search *(Find files without knowing their names)*
- 📚 AI-Powered Research across local documents
- 📄 Chat with PDFs, DOCX, PPTX, and TXT files
- 🧠 AI-generated document understanding and summaries
- ⚡ Streaming AI responses
- 📂 Automatic folder indexing
- 🔒 100% Offline & Privacy First
- 🖥️ One-Click Installer
- 🧩 Multiple Local LLM Support (Ollama)

---

## 🚀 Why AXON?

Traditional file search depends on filenames.

But in real life, we usually remember **what a document contains**, not **what we named it**.

AXON solves this problem by searching through the **actual content** of your files using AI.

Simply type what you remember, and AXON finds the right document instantly.

Since everything runs locally:

- ✅ No Internet Required
- ✅ No Cloud Dependency
- ✅ No API Costs
- ✅ Complete Privacy
- ✅ Fast Local Processing

## ⚙️ Installation

Getting started with AXON is simple.

1. Open the **Releases** section of this repository.
2. Download the latest installer for your operating system.
3. Run the installer.
4. Follow the on-screen installation steps.
5. During the first launch, AXON automatically configures the required dependencies and local AI components.
6. Once the setup is complete, you're ready to use AXON completely offline.
